Custom DOM Options

About Custom DOM

This extension traverses the DOM (both static and dynamically generated). Each node for which the predicate function returns true will be passed to the handling function.

In this page, you can write predicate and handling functions. Those functions receive one parameter, 'node', which is an element.

As an example, you could write a predicate :

return node.className === 'foo'
       && node.nodeName.toLowerCase === 'span';

And a handler :
node.remove();
